# Club Penguin Blizzard - Desktop Client

## Play Here -> https://blizzardcp.pw

---
## SETUP

First, Install [NodeJS](https://nodejs.org) with [NPM](https://npmjs.com)

Then, Clone this Repo
```
git clone https://github.com/BlizzardCP/Client.git
```

After that, install all the dependencies
```
npm install
```

## COMPILING

### Compile Temporarily for Testing:
```
npm start
```

### Compile for Windows (32 bit):
```
npm run build-win32
```

### Compile for Windows (64 bit):
```
npm run build-win64
```

### Compile for Linux (AppImage, 32 and 64 bits):
```
npm run build-linux
```

### Compile for macOS (Intel):
```
npm run build-mac
```
